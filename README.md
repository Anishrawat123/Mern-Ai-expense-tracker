💰 MERN AI Expense Tracker

A modern AI-powered Expense Tracker Web Application built using the MERN Stack.
This application helps users track daily expenses, analyze spending patterns, and get overspending alerts.

---

🚀 Features

✅ Add new expenses easily
✅ Delete expenses anytime
✅ View all expenses on the dashboard
✅ Category-based expense tracking
✅ Overspending alert system
✅ Clean and responsive UI
✅ Real-time expense updates

---

🧠 AI Feature

The application analyzes user spending and provides alerts when spending exceeds a certain threshold.

Example Alert:
⚠️ You are overspending on Food

This helps users control their expenses and manage their budget better.

⚙️ Installation

Clone the repository

git clone https://github.com/Anishrawat123/Mern-Ai-expense-tracker.git

Go to project folder

cd Mern-Ai-expense-tracker

Install backend dependencies

cd server
npm install

Install frontend dependencies

cd ../frontend
npm install

---

▶️ Run the Application

Start backend server

cd server
npm start

Start frontend

cd frontend
npm start

Application will run on

http://localhost:3000
